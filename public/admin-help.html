Documentation for Event Organizers and Administrators.

Management

Creating Events

1. The Floating Button: On the Dashboard, click the red "+" button in the bottom right corner.

2. Event Details: Ensure you set a valid "Date" and "Capacity".

Tracking

Verifying Attendance

• Scan Mode: Use the "Scanner" page to scan student QR codes at the door.

• Manual Entry: You can also manually mark students as present via the "Manage Events" table.

Reporting

Exporting Data

• Analytics Tab: Navigate to "Reports" to view school-wide stats.

• CSV Export: Click the "Export" button on any event card to download the attendee list.

System Administration

Admin Support

How do I revoke a student's access?

Go to the "Students" tab, search for the User ID, and click the "Edit" icon. Toggle status to "Suspended".

Can I edit an event after it starts?

Yes, but be careful modifying "Location" or "Time" if students have already checked in.
